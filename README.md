# CoronavirusSim
A COVID-19 simultation. Typescript, p5.js, Webpack.

### Installation
First clone using

```
git clone https://github.com/RossmacD/CoronavirusSim.git
```

Then navigate to into the repo and install the dependancies with 

```
npm i
```

To create a build run

```
npm run build
```
This will create a build in the dist folder


To run in development mode with hot reloading run

```
npm run start
```
